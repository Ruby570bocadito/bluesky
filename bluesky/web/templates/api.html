API REST

Documentación de la API REST de Bluesky Web Dashboard.

Endpoints Disponibles
MétodoEndpointDescripción
GET /api/status Estado del sistema, uptime, hardware
GET /api/modules Lista de todos los módulos cargados
GET /api/modules/<name> Detalle de un módulo específico
POST /api/modules/<name>/run Ejecutar un módulo (body: {"target":"...", "options":{...}})
POST /api/scan Iniciar escaneo (body: {"scanner":"device|services", "target":"..."})
GET /api/scan/status Estado del escaneo actual
GET /api/hardware Información del hardware Bluetooth
GET /api/sessions Historial de sesiones
GET /api/reports Lista de reportes generados
GET /api/reports/<file> Contenido de un reporte específico
GET /api/logs Logs del sistema (?since=N para logs desde índice N)
GET /api/config Configuración actual de Bluesky
Ejemplos cURL 
# Estado del sistema
curl http://localhost:5000/api/status | jq .

# Listar módulos
curl http://localhost:5000/api/modules | jq .

# Ejecutar módulo KNOB
curl -X POST http://localhost:5000/api/modules/knob/run \
  -H "Content-Type: application/json" \
  -d '{"target":"AA:BB:CC:DD:EE:FF"}'

# Iniciar escaneo
curl -X POST http://localhost:5000/api/scan \
  -H "Content-Type: application/json" \
  -d '{"scanner":"device"}'

# Ver hardware
curl http://localhost:5000/api/hardware | jq .
Ejemplos Python 
import requests
import json

BASE = "http://localhost:5000"

# Estado del sistema
r = requests.get(f"{BASE}/api/status")
print(r.json())

# Ejecutar escaneo
r = requests.post(f"{BASE}/api/scan",
    json={"scanner": "device"})
print(r.json())

# Ejecutar módulo
r = requests.post(
    f"{BASE}/api/modules/crackle/run",
    json={"options": {"EXECUTE": "true"}}
)
print(r.json())

# Ver resultados
r = requests.get(f"{BASE}/api/scan/status")
print(json.dumps(r.json(), indent=2))
