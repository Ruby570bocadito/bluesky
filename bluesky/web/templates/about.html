Acerca de Bluesky

Framework de auditoría Bluetooth tipo Metasploit — v{{ version }}

Descripción

Bluesky es un framework de seguridad Bluetooth de código abierto que implementa una arquitectura modular estilo Metasploit para auditoría, detección de vulnerabilidades y pruebas de penetración en dispositivos Bluetooth.

Compatible con Windows, Linux, Termux (Android) y WSL, Bluesky unifica 15+ módulos de ataque, 3 escáneres, 3 exploits, un escáner de vulnerabilidades integrado (13+ checks), consola interactiva REPL, dashboard web, y un autopilot automatizado de 4 fases (scan→detect→attack→report).

Arquitectura
  • Core Engine: Descubrimiento dinámico de módulos, carga y ejecución con duck-typing
  • 15 Módulos de ataque: KNOB, BIAS, BLUFFS, BlueBorne, BlueFrag, BTSpam y más
  • 3 Escáneres: Dispositivos, servicios SDP, vulnerabilidades (13+ checks)
  • 3 Exploits: Keystroke injection, L2CAP fuzzing, RFCOMM shell
  • Autopilot v2: Pipeline automático de 4 fases con detección dinámica
  • Consola REPL: Interfaz tipo Metasploit con search, check, back, tab-completion
  • Web Dashboard: Flask + Bootstrap 5 + REST API (16 endpoints)
  • Multi-plataforma: Windows, Linux, Termux, WSL con backends específicos

Hardware

Hardware recomendado para ataques activos:

  • CSR 4.0 — KNOB, BIAS, BLUFFS (~$5-10)
  • TP-Link UB500 + DarkFirmware — BLUFFS avanzado (~$12)
  • nRF52840 — BTLEJack, BLE sniffer (~$25)
  • Ubertooth One — BLE sniffing avanzado (~$120)

Sin hardware específico, Bluesky opera en modo escaneo/detección.
